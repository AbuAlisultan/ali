Практика №1

Сделайте свое навигационное меню:

- Каждое значение в меню это ссылка.

- Нажатие на Home:: ссылка на "https://ru.wikipedia.org/wiki/Lorem_ipsum" в нынешнем окне.

- Нажатие на News:: ссылка на "https://www.lipsum.com/" в новом окне.

- Нажатие на Contact:: ссылка на "#".

Идеи:

- Будет хорошей практикой применить знания списков <ul>.

- В качестве дополнения, можете изменить CSS свойство cursor на значение crosshair. Данное свойство принадлежит HTML тегу <a>.

Занятие №5. Псевдоклассы.

Псевдоклассы - селектор, который выбирает элементы, находящиеся в специфическом состоянии.

Список пройденных тем:

  • Общая структура HTML разметки.
  • Заголовки (h1-h6).
  • Тег текста (p).
  • Теги стилизации текста без CSS (b, i, u).
  • CSS - inline, internal, external.
  • CSS - селекторы (Класс, Id, HTML Тег).
  • CSS - форматирование текста и шрифта.
  • Единицы измерения. (px, rem, %)
  • Блочные элементы (div).
  • CSS - display.
  • CSS - ширина, высота.
  • CSS - margin.
  • CSS - padding.
  • CSS - border.
  • CSS - списки (ul, ol, dl).
  • HTML - Изображения (img).
  • HTML - Гиперссылка (a).
  • CSS - Задний фон (background).
